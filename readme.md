# URL Shortener

This project allows users to generate short URLs for long links. The application stores the original URLs in a MongoDB database and handles redirection efficiently using an Express-based API. The frontend is built with React and communicates with the backend via REST APIs.

### Features
 - Generate short URLs from long links
 - Redirect short URLs to original destinations
 - RESTful API for URL creation and redirection
 - Simple and clean user interface
